Cờ Cá Ngựa
==========

Requirement
-----------

* Mô phỏng bàn cá ngựa cho 2 - 4 người chơi
* Mô phỏng chuyển động nước đi và ăn quân
* Thay đổi được góc nhìn tùy ý.
